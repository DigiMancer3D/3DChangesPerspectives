# 3D Changes Perspectives: Visual Explainers & Companion Archive

**3D Changes Perspectives** is a multi-platform livestream show that explores difficult, mind-expanding, and often controversial ideas from unconventional angles.

This repository serves as the **official visual companion archive**: containing all the images, diagrams, explainer graphics, slides, and supporting visuals used in the episodes so viewers can study, reference, and go deeper after watching.

---

## 🎯 What’s Inside This Repo

- High-resolution **explainer images & diagrams** for every major topic discussed on the show
- Slide decks and key visual aids
- Episode-specific graphics (timelines, mind maps, comparisons, “before/after” perspectives)
- Raw source files (PSD/SVG/PNG) for community remixing and study
- Curated reference sheets for viewers who want to continue learning

Everything is organized by episode/series so you can quickly find the visual that blew your mind in a particular show.
